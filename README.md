# miasistentevirtual
Asistente Virtual Educativo para el Desarrollo del Lenguaje en Educación Preescolar , diseñado como apoyo pedagógico para fortalecer el reconocimiento de vocales y habilidades fonológicas en niños de nivel preescolar. El sistema fue utilizado en un estudio con 102 niños, demostrando mejoras significativas en el aprendizaje
